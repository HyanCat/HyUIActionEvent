# HyUIActionEvent

> A Respond Chain Library for UIResponder.

## Installation

Add library in your `Podfile`

	pod 'HyUIActionEvent'

Then run install

	pod install

## Usage

1. Firstly, import header file

		#import <HyUIActionEvent/HyUIActionCore.h>

2. Create an event at somewhere ( in `UIView` or `UIViewController` )

		self dispatch
